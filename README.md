# NekoEPUB 🐱📘

NekoEPUB 是一款專注於 **圖片與 ePub 之間轉換與最佳化** 的現代化桌面應用程式。  
它解決的是一個被長期忽視、但實際非常痛苦的需求：  
**如何把大量圖片，快速、正確、可控地變成高品質 ePub，以及反向處理與體積優化。**

---

## ✨ 核心功能

### 📱 四個主要功能標籤

#### 1️⃣ 圖片轉 ePub
- 拖放或選擇多張圖片
- 手動調整圖片順序
- 第一張圖片自動設為封面
- 使用原始檔名作為書名
- 支援雙頁掃描模式（漫畫／掃描書適用）
- 一鍵清空圖片列表

#### 2️⃣ ePub 轉圖片
- 提取 ePub 中的所有圖片資源
- 顯示書籍基本 Metadata
- 統一轉換為 PNG 格式，方便後續處理

#### 3️⃣ 壓縮 ePub
- 調整圖片品質（10%–100%）
- 選擇輸出格式（JPEG / WebP）
- 即時預估壓縮後檔案大小
- 視覺化顯示壓縮前後體積差異

#### 4️⃣ 批次轉檔 ⭐
- 一次處理多個資料夾
- 每個子資料夾自動轉換為一本 ePub
- 以資料夾名稱作為書名
- 顯示完整統計資訊
- 即時狀態追蹤
- 支援雙頁掃描模式

---

## 📖 雙頁掃描模式（正確實作）

NekoEPUB 並非簡單拼頁，而是**正確實作 ePub 固定版面規範**：

- `pre-paginated` 固定版面
- `spread: none` 單頁顯示
- 橫向最佳化（landscape orientation）
- 每張圖片獨立一頁  
✅ 不會出現常見的「四頁並排」災難

## 🧠 設計哲學

NekoEPUB 並不追求「什麼都能做」，  
而是專注把三件事做到極好：

1. 圖片 → ePub
2. ePub → 圖片
3. ePub 體積最佳化

沒有多餘選項，沒有魔法按鈕，  
只有**可預期、可理解、可控制的結果**。
