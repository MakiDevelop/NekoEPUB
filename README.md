NekoEPUB — 將內容轉為 EPUB，並把體積壓到剛剛好

NekoEPUB 是一款輕量、專注的內容轉換工具，
讓你能夠將多張圖片快速整理成 EPUB，
或從既有內容中抽取頁面，重新封裝為更小、更好保存的電子書檔案。

它不追求複雜的編輯功能，
只專心處理一件事：
把內容轉成 EPUB，並把體積壓縮到合理範圍。

核心功能

• 將多張圖片（PNG／JPG）整理並輸出為 EPUB
• 依圖片順序自動建立電子書頁面
• 抽取內容並轉換為標準 PNG 圖像
• 有效壓縮 EPUB 檔案體積，降低儲存與傳輸成本
• 保持輸出結果相容於主流電子書閱讀器

NekoEPUB 特別適合以下情境：
圖片型內容電子書製作、掃描文件封裝、圖文資料歸檔、
需要控制檔案大小的 EPUB 發佈與備份需求。

為什麼選擇 NekoEPUB

許多 EPUB 工具專注在「閱讀體驗」或「格式微調」，
但在實務上，你更常遇到的是這些問題：

檔案太大、不好傳
內容來自圖片、不適合文字編輯
只想快速封裝，不想進入複雜流程

NekoEPUB 的選擇很直接：

• 圖片就是內容
• 順序就是結構
• 壓縮就是價值

• 不需要登入
• 不依賴雲端
• 所有處理都在裝置本地完成
• 不修改原始素材，對隱私與資料安全友善

設計理念

NekoEPUB 的設計理念很單純：
格式是手段，不是目的。

當你需要把一組圖片變成可攜帶的電子書，
或只是想讓 EPUB 檔案更小、更好管理，
NekoEPUB 會安靜地把事情做好，然後退到一旁。
